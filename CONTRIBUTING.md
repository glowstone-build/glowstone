# Contributing to Glowstone

Thank you for your interest in contributing!

## Before contributing

By submitting a pull request, you agree to the terms of the Contributor License Agreement contained in `CLA.md`.

Only contributions from authors who agree to the CLA will be merged.

## Development

* Follow the project's coding style.
* Write clear commit messages.
* Include tests when appropriate.
* Keep pull requests focused on a single change.
